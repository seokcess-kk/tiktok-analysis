## Problem Statement

### Current State
현재 시스템에서 소재(크리에이티브) 관련 기능:
- ✅ 소재별 성과 지표 조회 (CTR, CVR, CPA, ROAS)
- ✅ 피로도 지수 계산 (fatigue-calculator.ts)
- ✅ 소재 점수 산정 (creative-scorer.ts)
- ✅ 크리에이티브 인사이트 생성 (insight-generator.ts의 `generateCreativeInsight`)
- ❌ **소재별 개별 인사이트/전략 부재**
- ❌ **소재 교체 시점 예측 및 전략 부재**
- ❌ **소재 성공 요인 분석 부재**

### Pain Points
1. **소재별 인사이트 미제공**: 전체 계정 인사이트만 있고, 개별 소재에 대한 인사이트 없음
2. **전략 연결 부재**: 소재 분석 결과가 실행 가능한 전략으로 연결되지 않음
3. **교체 타이밍 불명확**: 피로도 지수는 있지만 언제 교체해야 하는지 구체적 가이드 부족
4. **성공 패턴 미분석**: 고성과 소재의 공통 특성 분석 없음

### Goal
각 소재에 대한 개별 인사이트와 실행 가능한 전략을 자동 생성하여, 마케터가 소재 최적화 의사결정을 쉽게 할 수 있도록 지원
