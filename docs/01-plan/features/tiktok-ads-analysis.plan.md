## 1. 프로젝트 개요

### 1.1 목적
TikTok Ads 광고 소재의 성과를 심층 분석하고, **GPT-5.2 기반 AI 엔진**으로 실행 가능한 인사이트와 전략을 자동 제안하는 대행사용 **지능형 광고 최적화 시스템** 구축

### 1.2 핵심 가치 (Priority Order)
1. **🎯 AI 전략 제안**: 데이터 기반 자동 최적화 전략 생성 및 실행 권고
2. **🧠 지능형 인사이트**: GPT-5.2 기반 심층 분석 및 원인 진단
3. **🎨 소재 성과 분석**: 크리에이티브별 성과 추적 및 최적 소재 발굴
4. **📊 성과 모니터링**: 실시간 KPI 추적 (보조 기능)

### 1.3 차별화 포인트
- **단순 대시보드가 아닌 "AI 광고 컨설턴트"**
- 데이터를 보여주는 것이 아니라 **"무엇을 해야 하는지"** 알려주는 시스템
- 마케터의 의사결정 시간 80% 단축 목표

### 3.1 AI 프롬프트 전략

#### 인사이트 생성 프롬프트 구조
```
[Context]
- 계정 정보, 업종, 목표
- 최근 7일 성과 데이터
- 히스토리 패턴

[Task]
- 성과 분석 및 이상 탐지
- 원인 추론
- 개선 방안 도출

[Output Format]
- 핵심 인사이트 (3줄)
- 상세 분석
- 권장 액션
```

#### 전략 제안 프롬프트 구조
```
[Context]
- 현재 캠페인 구조

[Constraints]
- 예산 한도
- 운영 정책
- 클라이언트 요구사항

[Task]
- 최적화 전략 도출
- 예상 임팩트 계산
- 실행 우선순위 결정

[Output Format]
- 전략 요약
- 구체적 액션 아이템
```
