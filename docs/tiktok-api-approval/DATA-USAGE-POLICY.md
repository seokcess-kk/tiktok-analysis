# TikTok API 데이터 활용 정책

> **문서 버전**: 1.0
> **작성일**: 2026-02-10
> **용도**: TikTok Marketing API 승인 신청용

---

## 1. 데이터 수집 범위

### 1.1 수집하는 데이터
| 데이터 유형 | 설명 | API Scope |
|------------|------|-----------|
| 광고주 정보 | 계정 ID, 계정명, 시간대, 통화 | `advertiser.info.read` |
| 캠페인 데이터 | 캠페인 ID, 이름, 상태, 예산, 목표 | `campaign.read` |
| 광고그룹 데이터 | 광고그룹 ID, 이름, 타겟팅 설정 | `adgroup.read` |
| 광고 데이터 | 광고 ID, 이름, 상태, 소재 연결 | `ad.read` |
| 소재 데이터 | 소재 ID, 유형, 썸네일 URL | `creative.read` |
| 성과 메트릭 | 노출, 클릭, 전환, 비용 등 | `report.read` |

### 1.2 수집하지 않는 데이터
- 최종 사용자 개인정보 (PII)
- 사용자 행동 로그
- 결제/금융 정보
- 위치 정보

---

## 2. 데이터 활용 목적

### 2.1 주요 활용 목적
| 목적 | 설명 |
|------|------|
| 성과 분석 | 광고 성과 메트릭 시각화 및 분석 |
| 인사이트 생성 | AI 기반 성과 진단 및 개선점 도출 |
| 리포트 생성 | 정기 성과 리포트 자동 생성 |
| 최적화 제안 | 예산/타겟팅/소재 최적화 전략 제안 |

### 2.2 금지 활용
- 데이터 재판매 또는 제3자 공유
- 광고 목적 외 마케팅 활용
- 경쟁사 분석 목적 활용
- 사용자 프로파일링

---

## 3. 데이터 보관 정책

### 3.1 보관 기간
| 데이터 유형 | 보관 기간 | 근거 |
|------------|----------|------|
| 성과 메트릭 | 24개월 | 연간 비교 분석 필요 |
| AI 분석 결과 | 12개월 | 인사이트 히스토리 |
| 접근 로그 | 6개월 | 보안 감사 목적 |

### 3.2 데이터 삭제
- 보관 기간 만료 시 자동 삭제
- 클라이언트 요청 시 30일 이내 삭제
- 계정 연동 해제 시 90일 이내 삭제

---

## 4. 데이터 보안 조치

### 4.1 전송 보안
- TLS 1.3 암호화 통신
- HTTPS 필수 적용

### 4.2 저장 보안
- API 토큰: AES-256 암호화
- 데이터베이스: 암호화 at rest
- 백업: 암호화된 스냅샷

### 4.3 접근 제어
- 역할 기반 접근 제어 (RBAC)
- 클라이언트별 데이터 격리
- 감사 로그 기록

---

## 5. 제3자 공유 정책

### 5.1 공유 원칙
- 사용자 동의 없이 제3자 공유 금지
- 법적 요구 시에만 예외적 공유

### 5.2 하위 프로세서
| 서비스 | 용도 | 데이터 유형 |
|--------|------|------------|
| OpenAI GPT | AI 분석 | 익명화된 메트릭만 |
| AWS/Vercel | 호스팅 | 암호화된 데이터 |
