## 2. 신청 절차 (Step by Step)

### Step 1: 개발자 포털 접속
```
URL: https://developers.tiktok.com/
```
1. "Log in" 클릭
2. TikTok 계정으로 로그인 (없으면 새로 생성)
3. 개발자 약관 동의

### Step 2: 앱 생성
1. 대시보드에서 **"Manage apps"** 클릭
2. **"Create app"** 버튼 클릭
3. 기본 정보 입력:
   - **App name**: `TikTok Ads Analyzer` (예시)
   - **App icon**: 512x512px PNG
   - **Description**: 아래 템플릿 참조

### Step 3: 플랫폼 설정
1. **Platforms** 섹션에서 "Web" 선택
2. **Redirect URI** 입력:
   ```
   https://your-domain.com/api/auth/callback/tiktok
   ```
3. **Terms of Service URL** 입력
4. **Privacy Policy URL** 입력

### Step 4: 제품(Products) 추가
1. **"Add products"** 클릭
2. **"Marketing API"** 선택
3. 필요한 Scopes 선택:

| Scope | 설명 | 필수 |
|-------|------|------|
| `advertiser.info.read` | 광고주 정보 조회 | ✅ |
| `campaign.read` | 캠페인 데이터 조회 | ✅ |
| `adgroup.read` | 광고그룹 데이터 조회 | ✅ |
| `ad.read` | 광고 데이터 조회 | ✅ |
| `creative.read` | 소재 정보 조회 | ✅ |
| `report.read` | 리포트 데이터 조회 | ✅ |

### Step 5: 앱 상세 정보 입력
1. **Use Case Description** 작성 (아래 템플릿 사용)
2. **Business Verification** 자료 업로드
3. 모든 필드 검토

### Step 6: 심사 제출
1. **"Submit for review"** 클릭
2. 최종 확인 후 제출
3. 심사 기간: 수일 ~ 2주

## 3. 앱 설명 템플릿 (Use Case Description)

아래 내용을 복사하여 신청서에 붙여넣으세요:

```
[앱 개요]
TikTok 광고 성과를 분석하고 AI 기반 인사이트를 제공하는
대행사용 광고 최적화 플랫폼입니다.

[주요 기능]
1. 광고 성과 대시보드
   - 캠페인/광고그룹/광고 단위 성과 조회
   - 주요 KPI 모니터링 (Spend, CTR, CVR, CPA, ROAS)

2. 소재(Creative) 성과 분석
   - 소재별 성과 비교 분석
   - 소재 피로도 추적
   - Top/Bottom 소재 자동 분류

3. AI 인사이트 생성
   - 성과 이상 탐지 및 원인 분석
   - 최적화 전략 자동 제안
   - 자연어 리포트 생성

4. 자동 리포팅
   - 일간/주간/월간 리포트 자동 생성
   - PDF 다운로드
   - Slack/카카오톡 알림 연동

[데이터 활용 방식]
- 읽기 전용 (Read-only) 데이터만 수집
- 광고 성과 메트릭 분석 목적으로만 사용
- 사용자 개인정보 수집하지 않음
- 제3자에게 데이터 판매/공유하지 않음

[대상 사용자]
- 광고 대행사 내부 마케터
- 클라이언트 광고 계정 관리 담당자

[예상 연동 계정 수]
- 초기: 6-20개 광고주 계정
- 동시 사용자: 5-10명

[보안 조치]
- OAuth 2.0 인증
- API 토큰 AES-256 암호화 저장
- HTTPS 필수 적용
- 클라이언트별 데이터 격리
```

## 6. 승인 후 다음 단계

1. **API 키 저장**
   - App ID
   - App Secret
   - `.env` 파일에 안전하게 저장

2. **OAuth 플로우 구현**
   ```
   Authorization URL: https://business-api.tiktok.com/portal/auth
   Token URL: https://business-api.tiktok.com/open_api/v1.3/oauth2/access_token/
   ```

3. **개발 환경 설정**
   - Sandbox 모드로 테스트
   - Rate Limit 확인 (기본: 10 requests/second)
